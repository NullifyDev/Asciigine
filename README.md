# Asciigine
Hello! This is a terminal-based game engine. Designed to be a Game Engine in a form of a Console Application. The main gole is to make the engine for 2D Projects while still being as simple to use as possible while caring about performance so that almost any older device can still run it at a smooth enough rate.

The project is currently under development but any Stars or Watches will be greately appreciated as it shows me that the project has people interested.
I know that the Terminal Game Engines are already a thing. But this is something ive been thinking of doing for a while.

Spoilers: This game engine ill have a 3D Support somepoint in a Major update. ;)

Please keep yourself up-to-date on it by watching this repository and watch it grow :D

## Usage

Because this is under development, you only need to do
```sh
dotnet run new <ProjectName> //To make a new project
dotnet run run <PrijectName> //To run an existing project
dotnet run -v
           --Version //Display current Game Engine version
```
